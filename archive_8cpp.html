Compresses data from an input stream using Huffman coding.

Reads the entire input, constructs a frequency table, builds a Huffman code table, and writes the serialised code table followed by the encoded data.

Parameters
inInput stream containing the original data.
outOutput stream where the compressed archive is written.
modeCompression mode (SAVE_TO_RAM or READ_TWICE).
Exceptions
HuffmanArchiveIStreamErrorif an I/O error occurs while reading.
HuffmanArchiveOStreamErrorif an I/O error occurs while writing or flushing.
