Detailed Description

Stores a code table for symbols (0–255), each associated with a bit sequence.

Codes are stored as Bits objects. The table can be serialised to a compact bit representation and deserialised back.

Constructor & Destructor Documentation

CodesTable::CodesTable ( const Bits &  bits)
explicit

Deserialises the table from a bit sequence.

Expected format:

  • 8 bits: number of entries N
  • For each entry:
    • 8 bits: symbol
    • 8 bits: code length L
    • L bits: code

Entries with L=0 are skipped.

bitsThe bit sequence containing the table.
