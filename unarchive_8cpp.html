Decompresses a Huffman‑coded archive.

Reads the code table and the encoded bit stream from the input, reconstructs the Huffman tree, and writes the decoded bytes to the output stream.

Parameters
inInput stream containing the archive.
outOutput stream for the decompressed data.
Exceptions
HuffmanUnArchiveIStreamErrorif an I/O error occurs while reading.
HuffmanUnArchiveOStreamErrorif an I/O error occurs while writing or flushing.
HuffmanUnArchiveBrokenDataErrorif the archive structure is invalid.
